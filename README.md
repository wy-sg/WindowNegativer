# WindowNegativer

`WindowNegativer` is a .NET 6 WPF utility that shows a movable, resizable frame with a click-through overlay that displays the covered screen area in negative colors.

## Features

- Resizable frame window
- Click-through negative overlay
- Topmost display
- Simple WPF implementation

## Build

Open the solution in Visual Studio and build the project, or run:

`dotnet build`
